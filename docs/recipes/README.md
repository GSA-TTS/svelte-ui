# Svelte UI Recipes

This directory contains recipes and patterns for using Svelte UI components in common scenarios.

## Available Recipes

- [Form Validation with Zod](./form-validation-with-zod.mdx) - Integrate Zod schema validation with form components

## What are Recipes?

Recipes are practical, copy-paste examples that demonstrate how to accomplish specific tasks with Svelte UI components. They go beyond basic component usage to show real-world patterns and integrations.

## Contributing Recipes

If you have a useful pattern or integration you'd like to share, please:

1. Create a new markdown file in this directory
2. Follow the existing recipe structure (Overview, Installation, Examples, Best Practices)
3. Include complete, working code examples
4. Test all code examples before submitting
5. Update this README with a link to your recipe
